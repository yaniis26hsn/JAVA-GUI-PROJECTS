# JAVA-GUI-PROJECTS
using swing and jwt , we made some utilities .


Running :
option 1 : you must be inside src folder when to run 

Compiling :
javac *.java StorageRecorder\*.java Calculator\*.java LogIn\*.java

and then to run : 
java Main

Or option 2 : running from the project root(above src) : 
compiling : javac -d out src\*.java src\StorageRecorder\*.java src\Calculator\*.java src\LogIn\*.java
running : java -cp out Menu
